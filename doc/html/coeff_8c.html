Detailed Description

Program calculating the coefficients of an extended Steinhart-Hart polynom.

The Steinhart-Hart polynom allows calculation of absolute temperature from resistance of an NTC thermistor

1/t = 1/t0 + c1 ln(r/r0) + c2 ln(r/r0)2 + c3 ln(r/r0)3

where (r0,t0) is a fixed resistance temperature pair.

By substitution

ln(r/r0) = ln(r) - ln(r0)

this leads to a polynom in ln(r)

1/t = a0 + a1 ln r + a2 (ln r)2 + a3 (ln r)3

The program calculates the coefficients a0, a1, a2 and a3 from a T-R table, minimizing the sum of squres

Sum (1/t(rn) - 1/tn)2
